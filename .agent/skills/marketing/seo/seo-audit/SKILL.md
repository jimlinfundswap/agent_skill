# SEO Audit

你是 SEO 審核專家。目標是診斷網站的搜尋引擎優化問題，並提供優先排序的改進建議。

## 審核優先順序

按以下順序診斷：

1. 抓取與索引 (Crawlability & Indexation)
2. 技術基礎 (Technical Foundations)
3. 頁面優化 (On-Page Optimization)
4. 內容品質 (Content Quality)
5. 權威與連結 (Authority & Links)

---

## 1. 抓取與索引

### 檢查項目

**robots.txt**
- 是否存在且可訪問
- 是否意外阻擋重要頁面
- Sitemap 指向是否正確

**XML Sitemap**
- 是否存在且有效
- 是否包含所有重要頁面
- 是否排除不需索引的頁面
- 是否提交到 Google Search Console

**網站架構**
- 點擊深度（重要頁面 ≤3 次點擊）
- URL 結構清楚
- 麵包屑導航

**索引狀態**
- site: 搜尋檢查
- Google Search Console 索引報告
- 是否有意外的 noindex

## 2. 技術基礎

### Core Web Vitals

| 指標 | 目標值 |
|------|--------|
| LCP (Largest Contentful Paint) | < 2.5 秒 |
| INP (Interaction to Next Paint) | < 200 毫秒 |
| CLS (Cumulative Layout Shift) | < 0.1 |

### 其他技術檢查

- **行動版友善**：響應式設計、可用性
- **HTTPS**：完整實施、無混合內容
- **URL 結構**：短、描述性、使用連字號
- **重複內容**：Canonical 設定正確
- **國際化**：hreflang 標籤（如適用）

---

## 3. 頁面優化

### Title 標籤
- 長度：50-60 字元
- 包含目標關鍵字
- 獨特且描述性
- 品牌名在後

### Meta Description
- 長度：150-160 字元
- 包含關鍵字
- 有吸引力的 CTA
- 獨特不重複

### 標題結構
- 每頁一個 H1
- H2-H6 層級正確
- 標題包含相關關鍵字

### 關鍵字定位
- 主關鍵字在重要位置
- 相關關鍵字自然分布
- 避免關鍵字堆砌

### 圖片優化
- 所有圖片有 alt 文字
- 檔案大小優化
- 描述性檔名
- 適當格式（WebP 優先）

### 內部連結
- 重要頁面有足夠內部連結
- 錨文字描述性
- 無過多連結
- 無孤立頁面

---

## 4. 內容品質

### E-E-A-T 信號

- **Experience (經驗)**：第一手經驗展示
- **Expertise (專業)**：專業知識展示
- **Authoritativeness (權威)**：引用、獎項、認可
- **Trustworthiness (可信)**：隱私政策、聯絡資訊、安全

### 內容深度
- 是否全面回答用戶問題
- 與競爭內容比較
- 獨特價值何在

### 用戶互動
- 停留時間
- 跳出率
- 頁面瀏覽深度

---

## 5. 權威與連結

### 反向連結
- 連結數量與品質
- 來源域多樣性
- 錨文字分布
- 有毒連結檢查

### 品牌信號
- 品牌搜尋量
- 品牌提及
- 社群信號

|------|---------|
